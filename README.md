## Sobre o projeto

Este repositório reúne materiais e códigos do **quinto workshop do DataViz** promovido pelo **Laboratório de Estatística (LES) da Universidade Federal Fluminense (UFF)**, com o tema:

> **Dashboard no RMarkdown: Relatórios Interativos**

O objetivo foi apresentar uma introdução à criação de realatórios dinâmicos no RMarkdown, usando a biblioteca Flexdashboard com integração ao Shiny.

---

## Conteúdo

- Estrutura do **Flexdashboard** 
- Inserção de elementos no **dashboard**  
- Interatividade com **Shiny**
